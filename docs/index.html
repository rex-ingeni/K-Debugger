K-Debugger

모바일 앱(Android/iOS)에 붙어 로그·네트워크·상태를 실시간으로 들여다보는 데스크톱 디버깅 도구.

macOS 다운로드 최신 릴리스(.dmg) · 설치 후 자동 업데이트 지원

주요 기능

📜

실시간 로그

디바이스 로그를 스트리밍으로 수집하고 필터링해서 본다.

🌐

네트워크 검사

앱이 주고받는 요청·응답을 가로채 헤더·바디까지 확인한다.

🗄️

상태 · 저장소

앱의 Preferences·DB 등 내부 상태를 조회하고 SQL을 실행한다.

🧩

WebView 디버깅

Chrome DevTools Protocol(CDP)로 앱 안 WebView를 직접 디버깅한다.

🔌

디바이스 자동 연결

adb로 연결된 기기를 자동 감지하고 세션을 유지한다.

⬇️

자동 업데이트

새 버전이 나오면 앱 안에서 클릭 한 번으로 받아 교체·재시작한다.

설치 & 업데이트

1

다운로드 — 위 버튼으로 최신 .dmg를 받아 Applications로 드래그한다.

2

실행 — 서명 전 단계라 첫 실행은 앱을 우클릭 → 열기.

3

업데이트 — 새 버전이 있으면 ⚙ 설정 버튼에 뱃지가 뜨고, 설정 안 업데이트를 누르면 자동으로 교체·재시작된다.
